Sync macOS ssh config with iCloud

Posted on 2016-08-22 |

As a developer, I have to restore backups sometime because of the beta system.

Fortunately, most data was on server these days. Not only Safari, Photos, Calendar, Note, Reminders, but also other apps use iCloud storage. This was great progress, I only have to re-download apps from App Store and brew shell after reinstall the system.

But the setting of ssh, configs, etc., were really annoying, furthermore I may lose my private keys.

So I new a folder which named macOS in iCloud Drive folder, and copied all the files without ., For example, .gitconfig ~> gitconfig

In fact, any cloud drive (like Dropbox) can do it, but
considering privacy and this letter A Message to Our Customers written by Tim Cook

iCloud was my choice ;
