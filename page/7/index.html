Custom UIViewController Presentation Transitions with UIStoryboardSegue

The most popular way is UIViewControllerAnimatedTransitioning

In order to do that, you have to:

  • New a subclass which may called XXXTransitioning inherit from NSObject and implement two functions of protocol UIViewControllerAnimatedTransitioning

  • Then:

    1. New an instance of the XXXTransitioning in the ViewController
    2. Set transitioningDelegate = self
    3. Extension the ViewController with UIViewControllerTransitioningDelegate and implement

  • Do it again for dismiss

I noticed Storyboard Segue has a Custom Kind

That‚Äôs it, no more coding delegate and Extension. Simple and clean :)
