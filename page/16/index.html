How to manipulate Filco keyboard layout to Mac OSX

Posted on 2014-12-12 |

之前入了 Filco 红轴，游戏 编码 两不误

之前虽然用 System Preferences -> Keyboard -> Modifier Keys -> Select keyboard -> Opt-Cmd Cmd-Opt，可以把 Cmd-Opt 调成苹果的键位
但每次打星际都要再改一次，从上面的过程也能看出来很繁琐

至于 F1 - F12，根本就没办法，另外锁屏也是问题

皇天不负苦心人，终于找到了这个，而且还是免费的：
Karabiner

设置好 Opt-Cmd 后，又把 F1 - F12 改成了苹果的功能键
顺便把右下角倒数第二个无用的键 改成了 FN
然后又把 Ctrl_R FN 位置换了下
右上角的 Pa/Br 改成了 Power

然后在 MenuBar -> SettingList -> + 一个无键位修改的 Profile，比如叫 Game

以后如果玩游戏，点击状态栏的图标，一键切换成 Game 就行了
