Xcode plugins faild to be loaded after updated

Posted on 2015-06-16 |

You can type the command in terminal after Xcode updated

this will added your new Xcode UUID to DVTPlugInCompatibilityUUID

But that is a risky way, for instance my Xcode 6.3.2 always crash after I ran this command without brain.

So I had to open the folder ~/Library/Application Support/Developer/Shared/Xcode/Plug-ins and deleted some suspicious plugins.

Of course also you can check the tail -f /var/log/system.log, and found which plugin cause the Xcode crash exactly in ~/Library/Logs/DiagnosticReports/Xcode_$time_$macbookname.crash
